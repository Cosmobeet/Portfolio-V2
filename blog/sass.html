My thoughts on starting sass

March 25, 2022

Misconceptions

One of my concens holding me back from starting sass was thinking that i had to be great at css. Honestly i couldn't be more wrong, sass builds on top of your css knowledge and compiles into css. With sass you're able to incorporate variables and nesting among other features which can make creating a style sheet quicker and easier.

So when should you start sass?

I would recommend starting to learn sass as soon as you understand css properties and nesting. Nesting can be a great way to jump in and start getting more comfortable with sass before you tackle using variables and maps.
